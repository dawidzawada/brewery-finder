## Brewery Finder

  

Application which allows to find breweries in choosen city(United States). Created in [**React.js**](https://reactjs.org/) with help of [**OpenBreweryDB**](https://www.openbrewerydb.org/)

  

**Check the** [**LIVE DEMO**](https://dawidzawada.github.io/brewery-finder/)

  

------------

  

### About

  

Find breweries in any city of USA! Created with [**React.js**](https://reactjs.org), [**Styled-Components**](https://www.styled-components.com/), [**Axios**](https://github.com/axios/axios) and powered by [**OpenBreweryDB**](https://www.openbrewerydb.org/)
